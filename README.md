# Flight Price Prediction

A simple machine learning project to predict flight ticket prices based on journey details like departure time, arrival time, duration, and number of stops. Built using Python, scikit-learn, and deployed with Streamlit.


## Features
- Predict flight prices using multiple models: Linear Regression, Random Forest, and XGBoost
- Interactive web app built with Streamlit
- Handles various flight details like total stops, journey day/month, departure/arrival time, and duration


## Installation

1. Clone the repository:

git clone https://github.com/AayushAcharya12/flight_price_prediction_ml.git

2. Change directory to the project:

cd Project_2_Flight_Price_Prediction

3. Install required packages:

    pip install -r requirements.txt

4. Run the Streamlit app:

streamlit run app.py
Follow the web interface to select the model, enter flight details, and get predicted prices.
